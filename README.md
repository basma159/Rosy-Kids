# Rosy Kids 🩷🛍️

Rosy Kids is a responsive kids clothing e-commerce website built using HTML, CSS, Tailwind CSS, and JavaScript.

The project provides a simple shopping experience where users can create an account, log in, browse products, add items to their cart, manage quantities, and save favorite products.

## 🚀 Features

* User Registration and Login system using LocalStorage
* Display products dynamically using JavaScript
* Search products by:

  * Product name
  * Price
  * Color
* Add products to cart
* Remove products from cart
* Increase and decrease product quantity
* Calculate total cart price
* Add and remove favorite products
* Persistent data using LocalStorage
* Responsive design for different screen sizes
* Modern UI using Tailwind CSS

## 🛠️ Technologies Used

* HTML5
* CSS3
* Tailwind CSS
* JavaScript (ES6)
* LocalStorage
* Font Awesome Icons
